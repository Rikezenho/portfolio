# Henrique Kuwai's Portfolio

> My cleanish portfolio made with Nuxt and Vue.JS.

## Building

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# generate static project
$ yarn run generate
```
